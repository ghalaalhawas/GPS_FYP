## 📝 Questions to Consider Before Week 2

1. **Mobile Framework:**
   - Do you have JavaScript/TypeScript experience? → Consider React Native
   - Want best performance? → Consider Flutter
   - Have a Mac and limited time? → Consider iOS Native
   - No Mac? → Avoid iOS Native, choose React Native or Flutter

2. **Development Environment:**
   - What OS are you using for development?
   - Do you have Android Studio or Xcode installed?
   - Do you have Node.js installed (for React Native)?

3. **Project Scope:**
   - Will you target iOS, Android, or both?
   - What region will you use for initial testing? (Suggest: Oxford or Cambridge)
   - Do you have access to a physical device for testing?
